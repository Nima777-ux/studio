# **App Name**: LexiShift

## Core Features:

- Interactive Substitution Engine: Define custom character or word replacement rules in real-time, allowing for instant A to C mapping.
- Live Cipher Feedback: Visualize the transformation of input text to output text immediately as you type using a reactive live-update field.
- AI Language Architect: Use a tool to generate creative, theme-based encryption schemes or synthetic languages using generative AI.
- Rule Management System: Store and manage multiple transformation rules locally to switch between different encryption styles effortlessly.
- Reverse-Translate Mode: One-tap toggle to flip the logic of the substitution engine for seamless decryption.
- Output Share Utility: Copy transformed results or share them directly with pre-defined code phrases.

## Style Guidelines:

- Primary Color: Deep Electric Violet (#89fac0), chosen for its high-contrast vibrancy against dark themes, evoking a sense of mysterious digital encoding.
- Background Color: Obsidian Night (#302963), a heavily desaturated violet hue providing a premium dark-mode aesthetic suited for security-themed apps.
- Accent Color: Luminous Cyan (#828282), positioned to provide a technical and crisp contrast for interactive elements and key data labels.
- Headline and Body Font: 'Space Grotesk' sans-serif, selected for its modern, computerized, and precise appearance which reinforces the cryptographic theme.
- Sharp, geometric line icons from a technical library to mirror the machined, objective look of the 'Space Grotesk' font.
- A focused, modular card-based layout that prioritizes input-output proximity to minimize visual scanning.
- Fast-paced micro-interactions, such as a glitch or scanline effect when generating new AI-driven encryption keys.
