# LexiShift | Interactive Cipher Studio

A professional-grade character substitution and cryptographic architecture environment built with Next.js 15.

## Features
- **Substitution Engine**: Define custom character mappings.
- **Caesar Protocol**: Default +3 character shift (a→d, A→D).
- **Bidirectional Processing**: Seamlessly switch between Encryption and Decryption modes.
- **Terminal Swap**: Instantly move output results back to source for multi-pass processing.
- **Modern UI**: Built with ShadCN UI, Tailwind CSS, and Lucide icons.

## Getting Started
1. The app initializes with a standard +3 Case-Preserving Caesar Cipher.
2. Enter text in the **Source Terminal**.
3. Use the **Controls Bar** to toggle between ENCRYPT and DECRYPT modes.
4. Use **Swap Terminals** to perform iterative encryption or verification.

## Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS
- **Components**: ShadCN UI
- **Icons**: Lucide React
